Number Base Converter

This is my first ever mobile app that I made using Android Studio - Java. In the school we are focusing in web development, so then I explored the mobile development in the internet I found some very useful video turiol about android development using Java as programming language.The Idea of making this came to me when I was thinking of what to practice with my current knowlegde

First, I search on the internet on how to convert one number base to another. Second, is to think on how it should be handle in the program. Third, is to try making a sample program without using any gui and then when I achieve the desired result. I make the gui and implement the program that I just made.

The functions of the application as you can see in the video are:

  • Decimal - binary
  • Decimal - Octal
  • Decimal - hexadecimal
  • Binary - Decimal
  • Binary - Octal
  • Binary - hexadecimal
  • Octal- decimal
  • Octal - binary
  • Octal - hexadecimal
  • Hexadecimal - Decimal
  • Hexadecimal - binary
  • Hexadecimal - octal
